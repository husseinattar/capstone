# Udacity Blockchain Capstone

The capstone will build upon the knowledge you have gained in the course in order to build a decentralized housing product. 

## Setup
Run the following lines
`npm install`
`cd eth-contracts`
`truffle compile`


## Run Tests 
`truffle test`
